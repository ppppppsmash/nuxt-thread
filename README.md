## アプリ：nuxt-threads

#### セットアップ
```bash
# pnpm

pnpm run dev
pnpm run build

```

```
pnpm i @nuxtjs/supabase @nuxtjs/taiwindcss @pinia/nuxt @prisma/client @vite-pwa/nuxt nuxt-icon prisma uuid
pnpm i -D @pinia-plugin-persistedstate/nuxt 
```
